# 🧠 Libft – My Brain vs. The C Standard Library 💥

Welcome to my **Libft** repo! This is where I tried to *recreate the C standard library from scratch*, because why not suffer a little? 😅

This project is part of the **1337** (part of 42 School curriculum), and it’s all about building a custom, personal C library that I can use in future projects 💪

---

## 📚 What I Learned (aka What Broke Me)

- 🧹 Mastered cleaning memory leaks like a janitor with a vengeance (`malloc`/`free`)
- 🧶 String manipulation like a C ninja (`strlen`, `strchr`, `strdup`, etc.)
- 🧠 How memory really works (still gives me nightmares)
- 🧪 Creating, compiling, and using a `.a` static library like a real C wizard
- 📦 Writing a `Makefile` that’s smarter than me
